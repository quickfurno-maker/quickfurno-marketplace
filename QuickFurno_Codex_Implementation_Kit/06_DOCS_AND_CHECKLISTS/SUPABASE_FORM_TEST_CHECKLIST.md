# Supabase and Form Test Checklist

Submit one test entry for each form:

- Client enquiry popup
- Homepage CTA form
- Category page Get Matched form
- Vendor registration form

Verify in Supabase:

- Name saved
- Phone saved
- City saved
- Category/subcategory saved
- Requirement/message saved
- Source page saved if available
- Created date/time saved

UI checks:

- Success message appears
- Error message appears if required field missing
- Mobile keyboard does not break layout
- No console errors
- No duplicate submissions on double click
