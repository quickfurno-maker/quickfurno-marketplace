Before deployment:

- `git status` is clean
- `npm run build` passes
- Homepage desktop checked
- Homepage mobile checked
- Vendor listing desktop checked
- Vendor listing mobile checked
- Vendor landing page checked

- Vendor form submits to Supabase

- WhatsApp buttons work or show correct placeholder

- No horizontal mobile scroll
- Page speed acceptable
- SEO title/meta added
- Favicon/logo checked

- Open https://quickfurno.in
