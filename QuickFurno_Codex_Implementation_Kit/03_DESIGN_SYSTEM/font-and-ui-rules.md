# QuickFurno Font and UI Rules

Recommended production font pairing:
- Headings: Playfair Display, Cormorant Garamond, or Fraunces.
- Body/UI: Inter, Manrope, or Plus Jakarta Sans.

Safe default if you do not want to add new font packages:
- Headings: Georgia, serif fallback.
- Body/UI: Inter or system-ui.

UI rules:
- Homepage: no vendor cards on hero/homepage. Keep it client-first.
- Category page: compact vendor cards only.
- Vendor page: lead-growth focused, not client-focused.
- CTA color: copper/gold only.
- Background: warm ivory/cream, not pure white.
- Header/footer: dark charcoal.
- Form: popup/modal on CTA, not cluttered permanently on homepage.
- Animation: opacity + transform + hover lift. Avoid heavy 3D libraries for MVP.
